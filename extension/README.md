# LiveCode — live multiplayer coding

Edit a project together, live, with teammates and AI agents. No git push/pull.

## How to use

1. Open the folder you want to share (File > Open Folder).
2. Press `Ctrl+Shift+P` and run **LiveCode: Host a Session**.
   - A join link is copied to your clipboard. Send it to your friend.
3. Your friend opens an (empty) folder, runs **LiveCode: Join a Session**,
   and pastes the link.
4. You're now editing the same project live. Edits sync in ~1 second.

Run **LiveCode: Leave Session** to stop.

Works in VS Code, Antigravity, Cursor, and Windsurf.
